轻松拿 Windows 聚焦锁屏壁纸

原文始发于微信公众号（简Moments）：轻松拿 Windows 聚焦锁屏壁纸

在 Windows 10 中，当锁屏壁纸设置为  Windows 聚焦 时，锁屏之后就会看到一张漂亮的壁纸。这些壁纸会不定时自动更换，一般是一天换一张，不过有时候会看到重复的。

先来波图看一下 Windows 聚焦壁纸是不是真的很漂亮。

1920*1080

1080*1920

很漂亮吧，这还只是一小部分。今天我给大家分享一个小工具，只需要鼠标右键单击一下，所有的图片都会保存到你所指定的位置。不仅是这样，它还可以帮你分类，电脑手机都能用，nice。

工具我都打包好了，在公众号内回复“聚焦壁纸”即可获取下载链接。

如何使用

①鼠标右键单击，选择“使用 PowerShell 运行”

②等待操作执行完之后，按下回车，或者输入A，再按下回车

  • 图片将保持到这个位置C:UsersSTRxDesktopSpotlight，STRx是用户名，Desktop是桌面的意思，也就是文件夹在桌面，文件夹的名字叫Spotlight，打开之后里面会有2个或者3个文件夹

  • CopyAssets文件夹是执行完的图片缓存，可删除，进一步操作的时候会自动删除

  • Horizontal文件夹是1920*1080的图片，可用于电脑壁纸

  • Vertical文件夹是1080*1920的图片，可用于手机壁纸

③这样就拿到了现有的所有 Windows 聚焦 锁屏壁纸了，等下次看到新的喜欢的壁纸时，继续这样操作就行了

如何更改存储位置

①鼠标右键单击，选择“编辑”

  • 如图所示，框出来的就是图片的存储位置，可以随意更改，但是要统一。比如把DesktopSpotlight都改成DesktopSTRx

  • 注意图片必须在一个母文件夹中，例如：母文件夹是DesktopSTRx，那么图片的位置只能在DesktopSTRx后面加，比如DesktopSTRxA

②更改完保存退出，快捷键“Ctrl+S”
